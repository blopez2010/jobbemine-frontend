# jobbemine-frontend

Frontend repository to look for bios, opportunities and search for job opportunities

Demo: https://blopez2010.github.io/index.html

## Requirements

1. IDE: VS Code.
2. NodeJs Latest.
3. Git.

## Installation

Open the terminal and run the following commands:

1. Run `npm install yarn -g`.
2. Run `yarn`
3. Run `yarn start`.
